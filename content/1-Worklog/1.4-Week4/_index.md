## Week 4 Achievements

- Gained a solid understanding of Amazon S3 and the Object Storage model.
- Successfully created and managed Amazon S3 buckets and objects.
- Learned how to optimize storage costs using different S3 Storage Classes and Lifecycle Policies.
- Practiced enabling Versioning and Object Lock to improve data protection.
- Understood the role of AWS Storage Gateway in hybrid cloud architectures.
- Learned how AWS Snow Family supports large-scale data migration to AWS.
- Explored AWS Backup and common Disaster Recovery strategies, including Backup & Restore, Pilot Light, and Warm Standby.
- Learned the migration workflow using AWS Application Migration Service (MGN).
- Completed all storage, backup, and migration hands-on labs in the AWS First Cloud Journey program.
