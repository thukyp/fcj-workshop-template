# Worklog

Phần này ghi lại toàn bộ quá trình thực tập trong **12 tuần** tại chương trình **AWS First Cloud Journey**, từ **20/04/2026 đến 12/07/2026**.

Nội dung worklog phản ánh quá trình học tập, thực hành các dịch vụ AWS, tham gia workshop, thiết kế kiến trúc hệ thống, phát triển ứng dụng, triển khai, kiểm thử và hoàn thiện dự án **AWS Event Management Platform**.

Trong suốt quá trình thực tập, tôi từng bước tìm hiểu và áp dụng nhiều dịch vụ AWS như **Amazon Cognito, Amazon API Gateway, AWS Lambda, Amazon DynamoDB, Amazon S3, Amazon CloudFront, AWS WAF, Amazon SES, Amazon CloudWatch và AWS SAM** theo mô hình kiến trúc Serverless.

Các nội dung chính của từng tuần bao gồm:

- **Tuần 1:** Thiết lập môi trường học tập, tạo AWS Account, tìm hiểu Cloud Computing, IAM, Amazon VPC, Amazon EC2, Amazon RDS, Auto Scaling, Load Balancer và AWS Budgets.
- **Tuần 2:** Nghiên cứu Amazon VPC, Site-to-Site VPN, AWS Systems Manager, VPC Peering và Transit Gateway.
- **Tuần 3:** Thực hành quản trị Amazon EC2, Amazon CloudWatch, Auto Scaling Group, Load Balancer và Amazon Lightsail.
- **Tuần 4:** Tìm hiểu Amazon S3, Storage Gateway, AWS Backup, Disaster Recovery và AWS Application Migration Service.
- **Tuần 5:** Nghiên cứu AWS Security, IAM, AWS Organizations, Amazon Cognito, AWS KMS và AWS Security Hub.
- **Tuần 6:** Thực hành Amazon Redshift, Amazon ElastiCache, AWS Database Migration Service (DMS) và Schema Conversion Tool (SCT).
- **Tuần 7:** Xây dựng hệ thống phân tích dữ liệu với Amazon S3, AWS Glue, Amazon Redshift, Amazon QuickSight và Amazon DynamoDB.
- **Tuần 8:** Ôn tập kiến thức AWS, khảo sát đề tài, thiết kế kiến trúc hệ thống và khởi tạo dự án.
- **Tuần 9:** Thiết kế và phát triển module Attendance & Certificate, xây dựng chức năng QR Check-in bằng AWS Lambda, Amazon DynamoDB và API Gateway.
- **Tuần 10:** Hoàn thiện chức năng QR Check-in, tạo Certificate PDF, lưu trữ trên Amazon S3 và tích hợp quy trình Attendance.
- **Tuần 11:** Tích hợp các module của hệ thống AWS Event Management Platform, tối ưu kiến trúc Serverless, kiểm thử hệ thống và hoàn thiện tài liệu kỹ thuật.
- **Tuần 12:** Kiểm thử tổng thể, hoàn thiện báo cáo thực tập, cập nhật sơ đồ kiến trúc AWS, chuẩn bị demo và hoàn thành dự án.
