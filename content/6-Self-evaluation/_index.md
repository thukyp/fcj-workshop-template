During my participation in the **AWS First Cloud AI Journey (FCAJ)** program, I had the opportunity to contribute to the development of the **AWS Event Management Platform**, a serverless application built on Amazon Web Services.

Within the project team, I was mainly responsible for **Notification Management**, **Dashboard & Analytics**, **Cloud Deployment**, **System Monitoring**, and preparing the **Workshop Documentation**. In addition, I also participated in the research and implementation of several features in the **Attendance** and **Certificate** modules, while collaborating with other team members to refine the overall AWS architecture.

Throughout the project, I gained practical experience working with various AWS services, including Amazon CloudFront, Amazon SES, Amazon SNS, Amazon CloudWatch, Amazon EventBridge, Amazon S3, AWS Lambda, Amazon DynamoDB, Amazon Cognito, and AWS SAM. Besides implementing technical features, I also prepared workshop materials, wrote technical blogs, participated in FCAJ community events, and contributed to the project documentation.

Working on this project helped me gain a deeper understanding of how to build cloud-native applications using a serverless architecture. I also improved my knowledge of system design, service integration, deployment, monitoring, and collaboration between frontend, backend, and cloud services. Moreover, I strengthened my teamwork skills by using Git and GitHub for source code management, discussing technical solutions with teammates, and actively researching AWS documentation to solve technical challenges.

Participating in a real-world project also helped me better understand the software development lifecycle, the importance of testing, monitoring, continuous improvement, and effective collaboration throughout the development process.

Overall, the FCAJ internship provided me with valuable technical knowledge and practical experience, giving me a solid foundation to pursue my career as a **Backend Developer** and **Cloud Engineer**.
